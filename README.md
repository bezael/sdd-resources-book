## Cómo usarlos

**Para un proyecto nuevo:**

1. Empieza con `constitution-template.md` — captura las decisiones técnicas del proyecto
2. Usa `plantilla-prd.md` para cada feature que vayas a construir
3. Corre `checklist-calidad.md` antes de darle el ok al agente

**Para un proyecto existente:**

1. Pídele al agente que genere un draft de constitution leyendo el código actual
2. Revisa y corrige el draft
3. Aplica `plantilla-prd.md` a las próximas features (no intentes documentar todo lo que ya existe)
